


## Setup

To setup the app for development on your local machine, please follow the instructions below:

1. Clone the repo to your machine

   

2. Install packages

   If you use `npm`

   ```bash
   npm install
   ```

   or

   If you use `yarn`

   ```bash
   yarn
   ```

3. Run the development server

   If you use `npm`

   ```bash
   npm start
   ```

   or

   If you use `yarn`

   ```bash
   yarn start
   ```

4. Visit <http://localhost:3000>
